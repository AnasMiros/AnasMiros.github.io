# Portfolio Website

A clean, responsive portfolio website built with HTML, CSS, and JavaScript.

## Features

- **Responsive Navigation**: Sticky navbar with mobile hamburger menu
- **Dynamic Header**: Animated floating icons with gradient background
- **Interactive Sections**: About, Projects, and Contact sections
- **Creative Battle Scene**: Animated sword fight between "Creativity" and "Technology"
- **Project Cards**: Showcase for academic and personal projects
- **Contact Buttons**: Elegant email buttons with hover effects
- **Performance Optimized**: Clean CSS with modern layout techniques

## Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Flexbox, Grid, CSS animations, and transitions
- **JavaScript**: Interactive elements and animations
- **Font Awesome**: Icon library
- **Google Fonts**: Inter font family

## Setup

1. Clone or download the project files
2. Ensure all files are in the same directory:
   - `index.html`
   - `style.css`
   - `script.js`
3. Open `index.html` in any modern web browser

## Browser Support

Compatible with all modern browsers (Chrome, Firefox, Safari, Edge).
But there might be some visual glitches on Edge.
